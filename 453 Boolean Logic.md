| Rule           | Example                   |
| -------------- | ------------------------- |
| Identity       | x + 1 = 1                 |
| Null element   | x + 1 = x                 |
| Idempotency    | x + x = x                 |
| Involution     | (x')' = x                 |
| Compliments    | x + x' = 1                |
| Communtativity | x + y = y + x             |
| Associativity  | (x + y) + z = x + (y + z) |
| Distribution   | z(x + y) = zx + zy        |
| Covering       | x + xy = x                |
| Combining      | xy + xy' = x              |
| Consensus      | xy + x'z + yz = xy + x'z  |
| Demorgan       | (xy)' = x' + y'                          |
